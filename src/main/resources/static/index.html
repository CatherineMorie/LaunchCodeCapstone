Job Tracker

This application will allow users to add jobs applied for and track the status throughout the application and interview process. User will be able to search and view jobs by Company Name, Job Req Number or Job Status to start.

Stretch goal: Add login page to allow for different users to access platform and only show jobs added by the validated user. Reformat for mobile use.


Jobs will consist of the following information:
  • Company Name: Input
  • Job Req Number: Input
  • Position Title: Input
  • Salary: Drop-down List
  • Position Details: Text Area
  • Date Applied: Date Control
  • How Applied: Input with Dropdown
  • Job Status: Radio Buttons - Applied, Interview, Offer Accepted, Declined, No Longer in Consideration
  • User notes: Text Area

Reason for Creation:
There are numerous applications that allow users to search for and apply to positions including Indeed, CareerBuilder and LinkedIn. None of these allow the user to manage applications outside of their platform.
This allows users to track all known information in one place as well as keep their own notes and thoughts on a specific position in one place.
- Speak with the hiring manager and really love their personality?
- Get a dismissive brushoff from the Tech Leads or HR that makes you second guess moving forward?
These are all things you can note for comparing opportunities as you move through the process.
